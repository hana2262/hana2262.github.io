Prompt Distillery (提示词蒸馏器) v1.0发布啦！

发表于2026-02-27|更新于2026-02-27
|浏览量:

💡 起源：为什么要做这个工具？

在 AI 创作爆发的今天，我发现自己每天都在和各种 Prompt（提示词）打交道。但是，我的工作流却是一团糟：

  • 碎片化：好的提示词散落在微信聊天记录、Notion 笔记、本地 TXT 文档里，想用的时候根本找不到。
  • 版本混乱：同一个 Prompt 改了十几个版本，最后不知道哪个效果最好。
  • 重复劳动：每次都要手动复制粘贴，还需要人工去重、分类。

我试过很多市面上的工具，有的太重（功能复杂），有的必须联网（隐私担忧）。于是，我决定自己写一个简单但是我自己用着舒服的小窗口——Prompt Distillery (提示词蒸馏器)

核心理念：它不仅仅是一个存储工具，更是一个“蒸馏器”。通过无感监听和 AI 辅助，帮你把杂乱的灵感提纯为高质量的生产力资产。

核心特性：它能做什么？

1. 无感监听，自动捕获

不需要你手动打开软件，Prompt Distillery 会静默运行在后台。当你复制了一段疑似 Prompt 的文本时，它会自动识别意图并弹窗提示保存。

  • 支持正则匹配，智能过滤无关内容。
  • 点击通知直接归档，无需切换窗口。

2. AI 蒸馏，拒绝重复

这是我最喜欢的功能！借助本地算法和 AI 能力，它能：

  • 智能查重：当你存入相似提示词时，它会告诉你“已存储类似模板”。
  • 语义合并：把几条零散的修改意见，合并成一个完整的 Prompt。
  • 版本 Diff：保留近10条历史版本，直观展示优化前后的差异，保留每一次灵感迭代。

3. 分层管理体系

从“草稿箱”的灵感碎片，到“生产级”的成熟模板，一站式管理。

  • 多维度标签：支持自定义标签体系。
  • 变量支持：支持 {{keyword}} 语法，使用时动态替换变量。

4. 隐私优先 (Privacy First)

我知道 Prompt 是创作者的核心资产。

  • 纯本地存储：所有数据存储在本地 (LocalStorage/IndexedDB)，不上云。
  • 安全可控：支持敏感词过滤。

技术栈揭秘

作为一个开源项目，我也想分享一下它的技术实现：

  • 核心框架：Electron (主进程) + Vue 3 (渲染进程)
  • 构建工具：Electron Forge + Vite (秒级热重载，开发体验极佳)
  • UI 框架：Tailwind CSS (原子化 CSS，手写样式几乎为零)
  • 数据存储：Lowdb (轻量级本地 JSON 数据库)

项目采用了 三态窗口设计 (Mini悬浮窗 / 标准视图 / 沉浸编辑器)，为了实现这个功能，我在 Electron 的窗口管理上下了不少功夫，通过 BrowserWindow.setBounds 动态调整，保证了状态的连续性。

下载与体验

目前 v1.0.0 版本仅支持 Windows (x64) 系统（macOS 版本正在计划中）。

你可以前往 GitHub Releases 页面免费下载：

👉 点击下载 Prompt Distillery v1.0.0

  • 安装版 (.exe)：推荐普通用户下载，支持自动更新。
  • 便携版 (.zip)：绿色免安装，解压即用。

未来计划

  • 用户操作体验优化
  • 支持 WebDAV 云同步（坚果云/NAS用户福音）
  • 接入更多本地大模型及外部api（Ollama/DeepSeek）
  • 优化长文本的 Diff 算法
